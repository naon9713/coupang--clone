# Multi-Vendor Shopping Mall Development Checklist

## Phase 1: 프로젝트 설정 및 기본 구조

### 1.1 프로젝트 초기 설정
- [ ] Next.js 14 프로젝트 생성 (TypeScript, App Router)
  - **인수기준**: `npx create-next-app@latest`로 프로젝트 생성 완료, TypeScript 설정 확인
- [ ] TailwindCSS 설치 및 설정
  - **인수기준**: tailwind.config.ts 생성, 기본 스타일 적용 확인
- [ ] shadcn/ui 컴포넌트 라이브러리 설치
  - **인수기준**: shadcn/ui 초기화 완료, 기본 컴포넌트 사용 가능
- [ ] ESLint 및 Prettier 설정
  - **인수기준**: .eslintrc.json, .prettierrc 설정 완료, 코드 포맷팅 작동

### 1.2 프로젝트 구조 설정
- [ ] 라우트 그룹 구조 생성 (/customer, /vendor, /admin)
  - **인수기준**: app/customer, app/vendor, app/admin 디렉토리 생성 및 레이아웃 파일 존재
- [ ] 공통 레이아웃 및 컴포넌트 구조
  - **인수기준**: 루트 레이아웃, 공통 헤더/푸터 컴포넌트 생성
- [ ] 환경 변수 설정 (.env.local)
  - **인수기준**: DATABASE_URL, NEXTAUTH_SECRET 등 필수 환경 변수 정의

### 1.3 상태 관리 설정
- [ ] Zustand 설치 및 기본 스토어 설정
  - **인수기준**: Zustand 설치, 기본 스토어 생성 및 사용 예시 확인
- [ ] React Query (TanStack Query) 설치 및 설정
  - **인수기준**: React Query Provider 설정, API 쿼리 예시 작동
